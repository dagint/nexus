REST API Documentation

Nexus provides a versioned REST API for programmatic access to your job search data.

Authentication

All API endpoints require token authentication. Generate a token in Settings.

Include the token in the X-API-Token header:

curl -H "X-API-Token: YOUR_TOKEN" {{ request.host_url }}api/v1/pipeline

Alternatively, use Authorization: Bearer YOUR_TOKEN.

Rate Limiting

API requests are rate limited to 30 requests per minute per token.

Response Format

All responses follow a consistent JSON format:

{
  "data": [...],
  "meta": { "total": 42, "page": 1, "per_page": 20 }
}

Errors return: {"error": "Error message"}

Endpoints

GET /api/v1/jobs/search

Search for jobs across all configured providers.

ParameterTypeDescription
qstring (required)Search query
locationstringLocation filter
remote_onlybooleanFilter to remote jobs
pageintegerPage number (default: 1)
GET /api/v1/pipeline

List all jobs in your pipeline.

ParameterTypeDescription
stagestringFilter by stage (applied, screen, interview, offer, rejected, withdrawn)
POST /api/v1/pipeline/<job_key>/stage

Update the pipeline stage of a job.

Body: {"stage": "interview", "notes": "optional notes"}

GET /api/v1/bookmarks

List all bookmarked jobs.

POST /api/v1/bookmarks

Bookmark a job.

Body: {"job_key": "...", "title": "...", "company": "..."}

DELETE /api/v1/bookmarks/<job_key>

Remove a bookmark.

GET /api/v1/analytics

Get summary statistics for your job search.
