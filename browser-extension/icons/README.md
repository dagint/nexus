# Extension Icons

Place the following PNG icon files in this directory:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store / install dialog)

These should be square PNG images with transparent backgrounds. Use a briefcase or magnifying glass motif in the app's brand color (#75b798) on a dark background (#1a1a2e) for consistency with the main application.
